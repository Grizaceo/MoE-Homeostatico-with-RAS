# Resultados de experimentos

Outputs de ejecución de experimentos.

## Formato
Cada experimento crea un directorio con:
```
YYYY-MM-DD_HH-MM-SS_[nombre]/
├── config.yaml          # Configuración usada
├── metrics.json         # Resultados numéricos
├── rsi_log.jsonl        # Log RSI estructurado
└── plots/               # Visualizaciones
```
